**Krack Chat** is a real-time messaging platform designed for fast-paced conversations with a unique "Cracked Biscuit" aesthetic. It combines high-performance messaging with a playful, high-contrast dark theme.

## ‚ú® Key Features
 **Real-time Messaging:** Instant message delivery and receipt via Supabase.

 **Secure Auth:** Seamless user authentication and profile management.
 **Responsive Design:** Optimized for both desktop and mobile "snapping."

### Installation
   
   1. Clone the repository:
   
      git clone [https://github.com/Ram-git123/CHAT-APPLICATION-KRACK-CHAT-.git](https://github.com/Ram-git123/CHAT-APPLICATION-KRACK-CHAT-.git)
      
   3. Navigate to the project folder:
      
      cd krack-chat
      
   3.Install dependencies:
   
      npm install
      
   4.Configuration
   
      Create a .env file in the root directory and add your Supabase credentials:
      VITE_SUPABASE_URL=your_supabase_project_url
      VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
      
   5.Running Locally
     
     npm run dev
     
   6.vercel publish 
   
     https://chat-application-krack-chat.vercel.app/
